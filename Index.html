Site evaluation

Which site is it?

https://www.frankdeboosere.be
This is the personal website of Frank Deboosere.

Does it get regular updates?

The page has an automatic feed of Frank's Twitter and Facebook as well as the daily weather, which updates every day.
There is also a page about the 'bike diaries', which gets new entries a couple times a month.

How did we find it?

One of us already knew about it and told us that it was a really weird website.

Why does this website need an update/make-over?

Looks like a spam website, with the strange colours and overload of stuff on the homepage.
Because of this it's really hard to get a decent overview of the different pages.

Is this website active on social media?

Yes, Frank is very active on both Twitter and Facebook, which is probably why the site gets a surprising amount of visitors every day.
Especially knowing how badly the site is set up.

Top three things that should be changed.

  • Header needs to get its own background color, so it stands out more and is easier to use.
    The different pages and the order in which they're placed can use some major rethinking as well.
  • Change the entire colour scheme, because the black-white-blue does not work.
  • Most of the pages need to be rewritten, because it's difficult to find anything you're looking for.
    A lot of the information gets lost due to the messy way every page is set up.
